Outstar

Outstar network can associate a scalar input to a vector output for pattern recall. Learning algorithm is modified form of Hebbian learning rule.

 

In this configuration the from neuron activates a pattern of the to neurons

To create and train Outstar neural network with Neuroph Studio do the following:

  1. Create Neuroph project.
  2. Create Outstar network.
  3. Create training set
  4. Train network
  5. Test network

Step 1. Create Neuroph project click File > New Project.

 

Select Neuroph Project, click Next.

 

Enter project name and location, click Finish.

This created the project, next create neural network.

Step 2. To create Outstar network, click File > New File

 

Select project from Project drop-down menu, select Neural Network file type, click next.

 

Enter network name, select Outstar network type, click next.

 

Enter number of neurons in output layer, and click Finish button.

 

This will create the Outstar neural network with one neuron in input and eight in output layer.
