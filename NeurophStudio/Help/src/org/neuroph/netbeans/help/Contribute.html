HOW CAN I CONTRIBUTE

All kinds of contributions are welcome. You can:

  • Contribute code you've developed using Neuroph
  • Join the development by finishing some tasks from our todo list
  • Write documentation (improve this help)
  • Write about your experiments with Neuroph
  • Write benchmarks and compare Neuroph with other neural network software
  • Test Neuroph with some benchmarking data set from and publish your results on Neuroph wiki
